## Routes:

### Authorization Router:

#### -> POST /sign-up: 
    Header: NONE
    Body:   {
                "email": "******@*.com",
                "password": "**********" min=10
            }
    Return: NONE

#### -> POST /sign-in: 
    Header: NONE
    Body:   {
                "email": "******@*.com",
                "password": "**********" min=10
            }
    Return: Token
